# Contributing

Thanks for contributing to Smart Blob Engine.

## Principles

- Keep it dependency-free.
- Keep logic readable and testable.
- Prefer deterministic behavior over visual randomness where possible.

## Development notes

- Core engine: `script.js`
- Demo page: `index.html`
- Pathfinding uses graph + A* and should remain modular.

## Pull request checklist

- Keep changes focused and minimal.
- Explain behavioral changes in the PR description.
- Test in at least one Chromium browser.
- Ensure no broken console errors.
